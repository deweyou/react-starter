## React Starter

### Features

- React + Typescript + Vite + SWC
- Unocss(preset-icon) + Less
- Eslint + Stylelint + Prettier
- i18n + fontSplit
- react-router + classnames + lodash + usehooks + ...

### Usage

```bash
npx degit git@github.com:deweyou/react-starter#main --mode=git my-app

cd my-app

pnpm i
```
